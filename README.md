# brazilian-ids

A Python 3 package that provides functions and classes to validate several Brazilian IDs.

## Documentation

Current IDs:

- CNPJ
- Numeração única de processo judicial
- CEP
- Município
- CPF
- PIS/PASEP
- CNO
- SQL
